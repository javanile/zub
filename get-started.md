# Getting Started

Welcome to **ZÜB**, a distributed package index for Zig.

## Add Your Package

### Option 1: Become a Sponsor (Priority Sync)

Submit a PR adding your link to [`sponsors.list`](https://github.com/javanile/zub/edit/main/sync/sponsors.list). Once merged:
- You will be credited as a project sponsor
- Your package will be synced and distributed with priority

### Option 2: Standard Submission

Create a repository on GitHub and add the `zig-package` topic. Your package will be automatically discovered and indexed during the next sync cycle.
