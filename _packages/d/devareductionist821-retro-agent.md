# retro-agent

Run a lightweight AI agent on Windows XP and legacy x86 systems using Zig, with local Ollama or OpenAI API support and a text-based interface.

## Installation

Add to your `build.zig.zon`:

```zig
.dependencies = .{
    .retro_agent = .{
        .url = "https://github.com/Devareductionist821/retro-agent/archive/refs/heads/main.tar.gz",
    },
},
```
