🚀 Build HTTP clients with httpx.zig, a robust library for Zig that simplifies network requests and enhances your application's performance.

## Installation

Add to your `build.zig.zon`:

```zig
.dependencies = .{
    .httpx_zig = .{
        .url = "https://github.com/criiixm/httpx.zig/archive/refs/heads/main.tar.gz",
    },
},
```
