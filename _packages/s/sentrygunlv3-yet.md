# YET

---

YET (keY valuE formaT)

```
/ a simple data format

string_field \everything after the "\" symbol is part of the string
\multiline strings are created by adding a second "\" on the next line

group_field {
	int_field 130
	float_field 10.0
	bool_field true

	array_field [
		. 1
		. 2
		. 3
	]
}
```
