# engine

A simple GUI wrapper for building cross platform GUI apps.

## Installation

Add to your `build.zig.zon`:

```zig
.dependencies = .{
    .engine = .{
        .url = "https://github.com/loftafi/engine/archive/refs/heads/master.tar.gz",
    },
},
```
