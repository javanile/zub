# huggingface-xet

🗂️ Manage large files efficiently with XET protocol: compress, deduplicate, and store using Zig for optimized model and dataset handling.

## Installation

Add to your `build.zig.zon`:

```zig
.dependencies = .{
    .huggingface_xet = .{
        .url = "https://github.com/Lucas-kPaz/huggingface-xet/archive/refs/heads/main.tar.gz",
    },
},
```
