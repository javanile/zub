# resources

A minimalist zig module for collecting files into a resource bundle.

## Installation

Add to your `build.zig.zon`:

```zig
.dependencies = .{
    .resources = .{
        .url = "https://github.com/loftafi/resources/archive/refs/heads/master.tar.gz",
    },
},
```
