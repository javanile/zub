📊 Log and manage your application data efficiently with logly.zig, a simple logging library built for Zig, ensuring clarity and performance.

## Installation

Add to your `build.zig.zon`:

```zig
.dependencies = .{
    .logly_zig = .{
        .url = "https://github.com/Fantasm027/logly.zig/archive/refs/heads/main.tar.gz",
    },
},
```
